# Echipa-16-3IE25

Proiectul este realizat în limbajul de programare C#. Este o aplicație Windows Forms, ceea ce reiese din utilizarea spațiului de nume System.Windows.Forms și din faptul că aplicația pornește prin rularea unui formular principal (Form1). Programul configurează stilurile vizuale și setările de redare a textului, apoi inițiază bucla principală a aplicației cu Application.Run(new Form1()).

Astfel, proiectul este o aplicație desktop dezvoltată în C# folosind tehnologia Windows Forms.
